How is this site running?

Questions/comments? Feel free to contact me!

Golang Web App


Go HTTP Server

- A Golang HTTP server backend handling routing
- HTML, CSS, Javascript/JQuery on the front end
Big thanks to w3 schools for their HTML & CSS templates
HTML and CSS are NOT my strong suits, these templates helped make this process a lot easier

Docker

- Golang webserver hosted in docker container
- Other services, such as Authelia and Nginx Proxy Manager, also hosted in separate Docker containers
- Portainer running to more easily manage multiple containers
Check out this project on GitHub!

Hosting


Nginx Proxy Manager + Cloudflare

- Nginx Proxy Manager running in docker container on seperate Ubuntu Server VM
- NPM acts as reverse proxy, adding security and routing clients to the appropriate server
- Cloudflare DNS routes traffic to my IP as well as implementing security like DDoS protection
Find out more about Nginx Proxy Manager on their site

Authelia

- Authelia authentication server running in Docker container
- Paired with Ngninx Proxy Manager to handle authentication and authorization
- Can also implement Multi-Factor Auth
- Can be used for SSO for top level domains as well as all subdomains
Find out more about Authelia on their site

Server


Ubuntu Server VM

- Docker containers running on Ubuntu Server VM
- Nginx Proxy Manager running on separate Ubuntu Server VM
- All developed and configured remotely with Visual Studio Code docker server
- Remote maintenaince of Docker containers with Portainer
Check out VS code server for developing on remote servers
Check out Portainer for handling multiple Docker containers easily

Proxmox VE

- Both Ubuntu VMs (one personal one for this site!) hosted and managed inside Proxmox instance
- Proxmox also runs things for my home network, like PiHole

Mini PC

- Proxmox running bare metal on a mini-pc in my office
